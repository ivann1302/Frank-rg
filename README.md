# Простой блог

## Запуск

```bash
npm install
npm run dev
```

## Функционал

- Просмотр списка постов (заголовок, дата создания, дата редактирования)
- Создание нового поста
- Редактирование поста
- Удаление поста
- Просмотр полного текста поста

Данные хранятся в localStorage браузера.

## Язык

JavaScript (Svelte 5)
